Load Profile

A profile holds basic information about a mission such as:

Default waypoints and route

Default type and number of aircraft for the mission

Profiles, then, are very like templates encountered in word processing packages.

When a mission is authorized against a target, you will be prompted to select a mission profile that is appropriate. There are three sets of profiles available:
Standard

User: Strike

User: Patrol

You start with a set of Standard Profiles and these provide a basic mission set-up. The Save button on the Mission Profile Dialogue allows you to save your own profiles. This is useful when you have perfected an attack against a particular type of target and you want to use it on another target of the same type.

After a Standard Profile has been loaded in, the computer a.i. will adjust the number of aircraft allocated to the package to correspond to the size of the target and the surrounding defenses.

After a user-defined profile has been loaded in, the computer a.i. will make only limited adjustments:

B26 and B29 requirements are interchangeable.

The two F86 squadrons are also interchangeable.

In addition, for user-defined profiles:

The computer a.i. never adjusts the number of flights.

Fighter-Bomber squadrons are not interchangeable.
