Insert a Wave

This Dialogue is launched when the Insert Wave button is clicked on the Mission Dialogue.

You can use this Dialogue to define the main duty of the Wave and the Time over Target (ToT). It is possible to define a Time over Target that would result in the Wave arriving before the main Wave. This could be useful, for instance, if you want to do some flak suppression before your main Wave arrives.

There are three possible types of main duty that can be allocated to a Wave:

Bomb

A mission can consist of a number of Strike Waves. Each Wave can be allocated the main target for the mission or secondary targets can be introduced.

A Crack and Burn mission would consist of two Waves. These missions are useful against substantial warehouses that contain flammable stores. The first Wave would use iron bombs to crack open the store. The second Wave would use napalm to destroy the stores.

Flak Suppression

When an important target is heavily defended, it is reasonable to send out a Wave that arrives before the main strike force to suppress the flak.

Bar CAP

When an important target is located within the bob area of influence, it is reasonable to send out a Wave that arrives before the main strike force to set up a Combat Air Patrol which acts as a barrier between the strike force and the bobs.

When the main duty is Flak Suppression or Air Cover, it is still possible to have AAA Cover and Air Cover Groups in support. However this is unlikely to be an effective use of your resources.
