The Map

The mapscreen consists of a scrollable and zoomable map, one fixed set of icons and five dockable toolbars:
Title Toolbar

Main Toolbar

Utility Toolbar

Scale Toolbar

Filters Toolbar


On returning from a mission, you will be presented with a debrief stage which can be recognized in two ways:
The Title Toolbar will indicate that the current status is Debrief.

The Main Toolbar will consist of a modified set of icons.
