Dossier

This Dialogue displays details about the currently selected item on the map. There are three tabs:

Details

This report gives an overview of the current state of the item. MSR indicates with which of the Main Supply Routes the item is associated.

Damage

This report gives an up to date summary of the current Damage State of the item.

Notes

Notes are only available for the bigger targets.

Buttons

Center

Click on this button to center the map onto the item.

Zoom

Click on this button to zoom the map onto the item.

Photo

Click on this button to give a 3D view of the item.

Authorize

Click on this button to launch a mission against the item. The Load Profile Dialogue will open so that you can start the process of defining the mission. This button is disabled if the blue forces control the item.
